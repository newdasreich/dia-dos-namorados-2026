# Koromatch

Jogo interativo feito com Twine (Sugarcube) e HTML/CSS para um evento sazonal do servidor da Korokome.

A ideia é simples: você baixa um app de namoro e sai num date com algum personagem famoso.

---

## Sobre

* Evento sazonal da comunidade Korokome
* Feito especialmente para o servidor
* Meme total fizemos 1 dia antes da data
* O sistema de likes sorteia aleatóriamente entre todos que você deu like

---

## Como jogar

Caso o link não esteja mais ativo:
Abre o arquivo `.html` no navegador e vai clicando nas opções.
